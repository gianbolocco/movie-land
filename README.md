# Movie Land

Movie Land es una plataforma simple para explorar información básica sobre películas. Puedes buscar películas por palabra clave, y se mostrarán resultados con información como el título, año y categoría.

## Características

- **Búsqueda de películas**: Puedes buscar películas por título utilizando un término de búsqueda.
- **Información básica**: Para cada película, se muestra el título, el año y la categoría.
- **Interfaz limpia**: La aplicación tiene una interfaz sencilla que permite explorar películas de manera rápida y eficaz.

## Tecnologías usadas

- **React**: Para construir la interfaz de usuario.
- **CSS**: Para la estructura visual y el diseño.
- **OMDb API**: Se utiliza para obtener los datos de las películas.
  
## Instalación

Para empezar a usar Movie Land en tu máquina local, sigue estos pasos:

1. Clona este repositorio:
   ```bash
   git clone https://github.com/tuusuario/movie-land.git

2. Entra en el directorio del proyecto:
   ```bash
   cd movie-land

3. Instala las dependencias necesarias:
   ```bash
   npm install

4. Inicia la aplicación:
   ```bash
   npm start

La aplicación debería abrirse en tu navegador en http://localhost:3000.
